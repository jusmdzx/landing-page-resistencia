# Visibilidade Que Transforma 

Landing Page desenvolvida para apresentar um panorama real sobre as barreiras e a resistência da comunidade LGBTQIAPN+ no mercado de trabalho. O projeto foca em alto impacto visual, acessibilidade e design voltado para conscientização.

## 🎨 UI/UX Design

A interface foi projetada do zero com foco na estética "Dark Mode", utilizando tipografia brutalista e elementos neon para guiar a narrativa dos dados e marcos históricos.

* [Acessar Protótipo no Figma](https://www.figma.com/design/G8F7PqL2s5w4oGGXd2CwwV/landing-page-resistencia.?node-id=0-1&m=dev&t=wAIlKJCAU1u7JWX9-1)

## 💻 Tecnologias Utilizadas

O desenvolvimento foca em performance e estrutura semântica:
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Grid, Flexbox e Variáveis de estado)
* JavaScript Vanilla (Interseção de rolagem e interatividade)
